### Installations

For all installations attempt to use [homebrew](https://brew.sh/) 

#### Casks


```
brew install --cask aerospace google-chrome wezterm
```

#### Other Tools


```
brew install fzf git neovim nvm ripgrep tmux uv zoxide zsh-autosuggestions zsh-syntax-highlighting

```

Some notes if some are installed necessarily as dependencies:

- smart_workspace_switcher.wezterm
    - zoxide 
    - fzf
- telescope.nvim
    - ripgrep

### Other Instructions

- Enabled Aerospace to start on Login
    - Go to System Settings > General > Login Items & Extensions
    - Click "+" icon in Open at Login
    - Select "Applidations/Aerospace"
